# Deck Court Pilote

1. Probleme : les cabinets doivent simuler vite sans perdre la tracabilite.
2. Solution : evidence-first, calculs deterministes, sources officielles.
3. Demo : dossier Claire et Marc.
4. Confiance : data quality, limites de couverture, audit, revue humaine.
5. Workflow : dossier, simulation persistable, replay, rapport versionne.
6. Evidence : snapshots, hash, alertes de changement, diff de regles.
7. Gouvernance : RGPD, non-conseil, IA runtime desactivee.
8. Pilote : 1 cabinet, 3 cas fictifs, aucun conseil opposable.
