# Script Commercial 7 Minutes

## 0:00 - Positionnement

Ce n'est pas un chatbot fiscal. C'est une plateforme de simulation patrimoniale
et fiscale qui se concentre sur la preuve : calculs deterministes, sources
officielles, limites visibles et validation humaine.

## 0:45 - Cockpit cabinet

Ouvrir `/cabinet`. Montrer la completude du dossier, les donnees manquantes,
la data quality, le radar de vigilance, la timeline et les limites IFI.

## 2:00 - Simulation IFI

Ouvrir `/simulations`. Montrer la base IFI de 1,11 M EUR, puis le panneau
"Pourquoi ce resultat ?" : donnees utilisees, formule, calcul intermediaire,
source officielle, limites et action a valider.

## 3:15 - Evidence Center

Ouvrir `/evidence`. Filtrer Service-Public, montrer sourceVersion, hash,
regles liees, dernier controle et diff de regles.

## 4:15 - Comparateur

Ouvrir `/scenarios`. Comparer ne rien faire, donation simple, donation avec
reserve d'usufruit, arbitrage immobilier, cession partielle + reallocation.

## 5:30 - Workflow

Ouvrir `/workflow`. Creer un dossier demo, saisir un actif/passif, lancer une
simulation persistable, rejouer, enregistrer une revue expert, generer un
rapport versionne.

## 6:30 - Conclusion pilote

Ouvrir `/pilot`. Proposer un test avec 1 cabinet, 3 cas fictifs, aucune donnee
reelle au depart, puis branchement Postgres/Blob/auth quand le workflow est valide.
