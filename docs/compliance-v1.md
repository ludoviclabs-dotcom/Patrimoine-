# Conformite V1

## Registre RGPD

Les traitements pilotes sont :

- gestion de dossier patrimonial et fiscal ;
- stockage documentaire ;
- revue professionnelle et journal d'audit.

Les donnees visees sont patrimoniales, fiscales, familiales et professionnelles.
Aucune donnee reelle ne doit etre saisie sans tenant, mandat et consentement.

## AIPD

Une AIPD est consideree necessaire pour le pilote car le produit traite des
donnees hautement personnelles et peut produire des evaluations patrimoniales.

Mesures V1 :

- separation par tenant ;
- minimisation documentaire ;
- audit append-only ;
- validation humaine explicite ;
- pas d'IA runtime tant que le corpus source et le moteur metier ne sont pas
  stabilises.

## IA et AI Act

Le mode IA est prepare mais desactive. Avant activation :

- utiliser des structured outputs ;
- citer les sources officielles ;
- s'abstenir si preuve insuffisante ;
- tracer les prompts, sources et versions ;
- informer l'utilisateur lorsqu'une sortie est assistee par IA ;
- imposer la revue humaine avant tout livrable professionnel.
