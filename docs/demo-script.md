# Script Demo 7 Minutes

## 1. Ouvrir le dossier

Afficher `/dashboard`.

Message :

> Voici Claire et Marc, foyer dirigeant patrimonial. La plateforme ne donne pas
> un conseil fiscal definitif : elle prepare un dossier explicable et validable.

## 2. Lire la synthese

Montrer :

- patrimoine brut : 3 840 000 EUR ;
- dettes : 420 000 EUR ;
- patrimoine net : 3 420 000 EUR ;
- liquidites : 220 000 EUR.

Message :

> Le dashboard commence par les blocs patrimoniaux et les points a revoir.

## 3. Lancer IFI

Aller sur `/simulations`, scenario IFI.

Montrer :

- base simplifiee : 1 110 000 EUR ;
- seuil : 1 300 000 EUR ;
- conclusion indicative.

Message :

> La valeur n'est pas une phrase IA. C'est un calcul deterministe avec etapes.

## 4. Ouvrir les calculs

Montrer le tableau :

- residence principale : 900 000 x 70 % = 630 000 ;
- immobilier locatif : 600 000 ;
- SCI : 300 000 ;
- dettes : -420 000 ;
- base : 1 110 000.

Message :

> Chaque etape garde sa regle, sa source et son statut.

## 5. Montrer Evidence Center

Aller sur `/evidence`.

Message :

> Les sources officielles sont versionnees, datees et reliees aux regles.

## 6. Montrer transmission et facture electronique

Revenir sur `/simulations`, changer de scenario.

Message :

> Les modules non chiffrables en V0 restent sous forme de checklist et de score
> de preparation, sans pretendre trancher juridiquement.

## 7. Exporter le rapport

Aller sur `/report`, cliquer `Imprimer / PDF`.

Message :

> Le livrable separe hypotheses, calculs, sources, limites et questions ouvertes.
